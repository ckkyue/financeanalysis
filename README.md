# FinanceAnalysis

FinanceAnalysis is a Python research toolkit for stock screening, market analysis, and technical/fundamental factor studies. It combines data from `yfinance`, TradingView, Finviz, and Macrotrends to produce ranked stock lists, Excel exports, and supporting charts.

## What It Does

- Screens stocks using relative strength, trend, volatility, and earnings/fundamental filters.
- Supports both US and Hong Kong market workflows.
- Generates Excel outputs with extra formatting for easier review.
- Produces market breadth, sector rotation, VIX, MACD, volume profile, and price charts.
- Caches downloaded price and fundamentals data locally to reduce repeated requests.

## Main Entry Points

- [Program/stock_screener.py](Program/stock_screener.py) runs the stock selection workflow and exports screened candidates.
- [Program/market_analysis.py](Program/market_analysis.py) runs broader market and sector analysis plus chart generation.

The remaining Python modules in [Program/](Program/) provide shared helpers for data access, technical indicators, fundamentals, and plotting.

## Repository Layout

- [Program/](Program/) - core Python code, notebooks, and research data files.
- [Price data/](Price%20data/) - cached market price CSVs generated by the scripts.
- [Fundamentals/](Fundamentals/) - cached fundamentals and earnings CSVs.
- [Result/](Result/) - screening outputs, figures, and other generated results.
- [Backtest/](Backtest/) - backtest outputs and historical screening results.

## Setup

Clone the repository and install dependencies:

```bash
git clone --depth 1 https://github.com/ckkyue/FinanceAnalysis.git
cd FinanceAnalysis
pip install -r requirements.txt
```

If you see issues with market data downloads, update `yfinance`:

```bash
pip install --upgrade yfinance
```

## Typical Workflow

1. Run [Program/stock_screener.py](Program/stock_screener.py) to build ranked stock lists and Excel reports.
2. Run [Program/market_analysis.py](Program/market_analysis.py) to generate sector rotation and market breadth outputs.
3. Review files in [Result/](Result/) and cached data in [Price data/](Price%20data/) and [Fundamentals/](Fundamentals/).

## Data Sources

- `yfinance` for historical price data and some stock metadata.
- TradingView via `tvDatafeed` for alternate price retrieval.
- Finviz for summary fundamentals.
- Macrotrends and AlphaQuery for historical fundamentals and earnings dates.

## Notes

- Several scripts create or overwrite CSV and Excel files in the data and result folders.
- Some notebooks and MetaTrader source files in [Program/](Program/) are exploratory or auxiliary rather than part of the main Python workflow.
